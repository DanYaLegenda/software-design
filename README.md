# SDS_2023

Документация к ЯП SDS_2023 для курсового проекта КПО

| Типы переменных | размер |
| --- | --- |
| int | 4 byte |
| str | 1 byte++ |
| script | 0 byte |

script - значение, типо void (как в C++) для определения функции без возвращаемого значение
``
script helloWorld() [
  print "Hello World!";
  newline;
]
``
